# Ecommerce App

An E-Commerce Web Application built using the **MERN Stack (MongoDB, Express, React, Node.js)**.
Users can browse products, add them to the cart, view the cart summary, and proceed to checkout with all data synced to MongoDB.

## Tech Stack

- **Frontend**- React, Redux Toolkit, TailwindCSS, Axios
- **Backend**- Node.js, Express.js, MongoDB
- **Other Tools**- React Hot Toast, Postman

## Features

- Browse all products
- Add / Remove items from cart
- Cart quantity updates dynamically
- Checkout summary with total price

## Setup Instruction

### Clone Repository

```bash
git clone https://github.com/krishnam98/ECOM_App

cd ECOM_App
```

### Setup Backend

```bash
cd backend
npm install
```

### Create `.env` file in /backend

```
MONGODB_URI=your_mongodb_connection_string
PORT=5000
```

### Run Backend

```bash
npm Run dev
```

### (Backend will run at ` http://localhost:5000`)

### Setup Frontend

```bash
cd ../frontend
npm install
```

### Run Frontend

```bash
npm run dev
```

### (Frontend will run at `http://localhost:5173`)
